Funkcje aplikacji
Dodawanie zadań: Możliwość wpisania nowej treści zadania.
Status wykonania: Kliknięcie w zadanie zmienia jego status (np. przekreślenie).
Usuwanie: Możliwość trwałego usunięcia zadania z listy.
Zapisywanie danych: Dzięki użyciu bazy danych, zadania nie znikają po odświeżeniu strony.

Wykorzystane technologie
Python – logika aplikacji.
Flask – mikro-framework webowy.
Flask-SQLAlchemy – obsługa bazy danych SQL.
HTML/CSS – warstwa wizualna i struktura strony.
SQLite – lekka baza danych przechowywana w pliku.

Jak to uruchomić?
Sklonuj to repozytorium na swój komputer.
Zainstaluj wymagane biblioteki: pip install flask flask-sqlalchemy.
Uruchom plik główny: python app.py.
Wejdź w przeglądarce pod adres: http://127.0.0.1:5000.

Have fun
